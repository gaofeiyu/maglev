# Step 2: Adaptive Curriculum (自适应课程)

## 目标
基于 Profiling 结果，推送差异化内容。
(Mode: **Deep**)

## 课程逻辑

### Track A: The Apprentice (新手 / User)
*   **Goal**: "Safe Onboarding"
*   **Modules**:
    1.  **Protocol 101**: 为什么 Spec > Code？
    2.  **The Flow**: Issue -> Atomizer -> Quick Spec -> Code。
    3.  **Safety**: 为什么要用 `cross-validate`？
*   **Action**: 推荐阅读 `README.md`，并尝试运行一次 `/maglev-quick-spec` (Dry Run)。

### Track B: The Veteran (老手 / User)
*   **Goal**: "Efficiency"
*   **Modules**:
    1.  **Advanced Skills**: `maglev-design-ux`, `maglev-research`。
    2.  **Shortcuts**: 如何跳过繁琐步骤 (Direct Intent)。
*   **Action**: 介绍最新引入的 Design & Insight 技能。

### Track C: The Architect (Maker)
*   **Goal**: "Evolution"
*   **Modules**:
    1.  **Skill Forge v2**: 如何制作 Deep Skill。
    2.  **Atomizer Logic**: 路由是怎么工作的？
*   **Action**: 挑战任务："尝试用 Skill Forge 制作一个 Hello World 技能。"
