# Step 4: Handoff (交付)

## 目标
将设计思考固化为 `02_ui_design.md`，供开发和测试使用。

## 动作
1.  **Assemble**: 将 Persona, Journey, State Diagram 组合。
2.  **Write**: 写入 `specs/20_evolution/active/{slug}/02_ui_design.md`。
3.  **Review**: 询问用户是否满意，或需要调整。

## 输出结构
```markdown
# UI/UX Design: {Feature Name}

## 1. Empathy (同理心)
[Persona Description]

## 2. User Journey (用户旅程)
[Mermaid Journey Diagram]

## 3. Visual Logic (视觉逻辑)
### {Component A}
[State Diagram]

## 4. Design Details (设计细节)
- **Colors**: ...
- **Typography**: ...
- **Micro-Interactions**: ...
```
