# Tutor Workflow

**Goal**: 用户理解 Maglev 并在项目中找到方向。

## 流程 (Process)

### 1. Profile (Step 01)
- 破冰。
- 询问用户背景 (Role & Experience).
- 确定学习目标 (Overview vs Deep Dive).

### 2. Curriculum (Step 02)
- 根据 Profile 生成个性化课程表。
- 课程表包含：核心概念、必读文档、关键目录。

### 3. Guided Tour (Step 03)
- 交互式导览。
- 逐个讲解 Curriculum 中的 Item。
- 配合 File Read 展示实际文件内容。

## 启动
1. 阅读 `references/step-01-profile.md` 开始会话。
