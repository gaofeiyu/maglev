# Product Brief: Dark Mode for Maglev Dashboard

## Context
Users are complaining about eye strain during late-night coding sessions.
Maglev currently only supports a "Light Mode" (White background).

## Goal
Implement a system-wide Dark Mode that:
1.  Automatically detects OS preference.
2.  Allows manual toggle in Settings.
3.  Persists user preference.

## Constraints
*   Must use CSS Variables.
*   Must support Tailwind Dark Mode.
*   Launch: Q2 2026.
